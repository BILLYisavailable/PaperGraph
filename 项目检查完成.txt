四、如何启动项目（简要步骤）
================================================================

第1步：安装前置软件
-------------------
• Python 3.9+ ✓ (已安装 3.10.5)
• MySQL 8.0+  (需要安装)
• Neo4j Desktop (需要安装)
• Redis/Memurai (需要安装)

第2步：配置环境
-------------------
运行：setup.bat

第3步：配置数据库密码
-------------------
1. 复制：copy .env.example .env
2. 编辑 .env 文件
3. 修改 MYSQL_PASSWORD 和 NEO4J_PASSWORD

第4步：创建MySQL数据库
-------------------
mysql -u root -p
CREATE DATABASE paper_kg CHARACTER SET utf8mb4;
EXIT;

第5步：启动Neo4j
-------------------
打开Neo4j Desktop，创建并启动数据库

第6步：初始化数据库
-------------------
venv\Scripts\activate
python scripts\init_database.py
python scripts\load_sample_data.py

第7步：启动服务
-------------------
窗口1: start_server.bat
窗口2: start_celery.bat (可选)

第8步：访问系统
-------------------
http://localhost:8000/docs

七、API接口
================================================================

图谱接口：
• GET  /api/v1/graph/root - 获取根节点
• GET  /api/v1/graph/children/{id} - 获取子节点
• GET  /api/v1/graph/node/{id} - 节点详情
• POST /api/v1/graph/layout/persist - 保存布局

统计接口：
• POST   /api/v1/statistics/query - 查询统计
• DELETE /api/v1/statistics/cache - 清除缓存

导出接口：
• POST /api/v1/export/file - 创建导出任务
• GET  /api/v1/export/job/{id} - 任务状态
• GET  /api/v1/export/download/{id} - 下载文件
